val diff : (t -> t) -> t -> t
diff f x returns the exat derivative of a function f : scalar -> scalar at

point x. Simply calling diff f will return its derivative function g of the same type, i.e. g : scalar -> scalar.

Keep calling this function will give you higher-order derivatives of f, i.e. f |> diff |> diff |> diff |> ...
